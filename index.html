Answer:
  • Help
    • Operation List
      • Addition: + (e.x. "5+5" will return 10)
      • Subtraction: - (e.x. "3-2" will return 1)
      • Multiplication: *, x, or × (e.x. "4x3" will return 12)
      • Division: / or ÷ (e.x. "8/4" will return 2)
    • How to use
      • Enter something in the box, then press the enter key on your keyboard.
      • There is also a button to do this, which is really good for mobile users.
      • Something like "2+", "+7", or "2++7" will not work.
      • An example of what you should put is "2+7".
    • Changelog
      • v0.1: There is addition, subtraction, multiplication, and division. The limit is 10↑1,0009e15, AKA 10{1,000}9e15 or {10, 9e15, 1000}.
